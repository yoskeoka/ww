Place active execution plans here.

Use non-numeric, descriptive kebab-case filenames that match the branch description.
For example, branch `plan/remove-docker-from-integration-tests` maps to
`docs/exec-plan/todo/remove-docker-from-integration-tests.md`.

When a plan is completed and the PR is ready, move the file to `../done/` without
renaming it into a numeric convention.
